# Natural Language Processing & Sentiment Analysis

## Soft Nexis Technology - Data Science & ML Internship

### Project Overview

This project performs Sentiment Analysis on the IMDB Movie Reviews Dataset using Natural Language Processing (NLP) techniques and Machine Learning.

## Features

- Text Cleaning
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Vectorization
- Logistic Regression Classifier
- Model Evaluation
- Word Cloud Visualization
- Feature Importance
- Model Comparison

## Bonus Features

- 3-Class Sentiment Analysis
- Real-Time Sentiment Analyzer
- BERT Sentiment Analysis (Hugging Face)

## Technologies Used

- Python
- Pandas
- NLTK
- Scikit-Learn
- WordCloud
- Matplotlib
- Seaborn
- Transformers (BERT)

## Output Files

- top_words_by_sentiment.png
- sentiment_confusion_matrix.png
- wordcloud_positive.png
- wordcloud_negative.png
- feature_importance_nlp.png
- nlp_model_comparison.png
- sentiment_model.pkl

## Dataset

The project uses the IMDB Movie Reviews Dataset (50,000 reviews).

Download it from Kaggle:

https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

After downloading, place **IMDB Dataset.csv** in the project folder before running the notebook.
